# React + TypeScript + Vite + Go lang backend with Mongo DB

## Introducing todos list app with CRUD actions

### For GO fiber package is used
`go get github.com/gofiber/fiber/v2`

and Air for hot reload:

`go install github.com/cosmtrek/air/v2/cmd/air@latest`

### For typescript, chakra UI components are used:

`npm i @chakra-ui/react @emotion/react @emotion/styled framer-motion`

### using useMutation and useQuery from react
